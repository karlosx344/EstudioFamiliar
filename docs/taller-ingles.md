# 🛠️ Vocabulary in the Workshop

Aprende el nombre de tus herramientas para tus manualidades.

| Herramienta | Inglés | Pronunciación |
| :--- | :--- | :--- |
| Martillo | **Hammer** | /hám-er/ |
| Pegamento | **Glue** | /glu/ |
| Tijeras | **Scissors** | /sís-ors/ |
| Cinta métrica | **Tape Measure** | /teip mé-shur/ |

:::info Reto de Vocabulario
Pide a Papá o Mamá que te pregunten estas palabras. Por cada una que sepas, ¡gana 20 puntos!
